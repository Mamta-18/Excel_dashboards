# Excel Dashboard Projects 📊

Welcome to my **Excel Dashboard Projects Repository** 🚀  
This repository contains interactive and business-oriented Excel dashboard projects created using **Microsoft Excel, Pivot Tables, Charts, Power Query, and Data Visualization techniques**.

These projects focus on transforming raw business data into meaningful insights through clean dashboard designs and analytical reporting.

An interactive Excel dashboard developed to analyze Blinkit grocery sales data, customer purchasing behavior, product performance, and revenue insights.

- Product category analysis
